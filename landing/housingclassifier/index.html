Housing Market Classifier

Norwegian Property Market Risk Prediction with Machine Learning
~ * ~ * ~ * ~ * ~ * ~ * ~ * ~ * ~

What is this?

This project classifies the Norwegian housing market into three states: HOT (prices rising >2%), STABLE (prices roughly flat), and COOLING (prices dropping >0.5%).

It uses real economic data from Statistics Norway (SSB) including property price indices, policy interest rates, CPI, population changes, and property sales volumes across 8 Norwegian regions.

How does it work?

The Random Forest classifier is implemented from scratch in Python — no scikit-learn for the core algorithm! It builds 50 decision trees, each trained on a bootstrap sample of the data, using Gini impurity for splits.

The model uses engineered features like price momentum, rate-price interactions, supply/demand imbalance scores, and regional strength indicators. Data is split chronologically (65/20/15) to respect the time-series nature.

Data Sources

Dataset Source Period
Property Price Index SSB Table 07221 2020-2024 (quarterly)
Policy Rate SSB Table 10701 2020-2024 (monthly → quarterly)
Consumer Price Index SSB Table 03013 2020-2024 (monthly → quarterly)
Population Change SSB Table 01222 2020-2024 (quarterly)
Property Sales Volume SSB Table 10187 2020-2024 (quarterly)

Tech Stack

  • Python 3.12+
  • Random Forest (built from scratch!)
  • Pandas & NumPy for data processing
  • Matplotlib & Seaborn for visualization
  • scikit-learn (only for evaluation metrics)

Regions Covered

Oslo, Rogaland, Vestland, Trondheim, Innlandet, Agder, Nordland, and More og Romsdal
